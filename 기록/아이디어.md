## 프로젝트 구상 이유
- 친구들과 공유 다이어리 앱을 쓰고 있었는데 같이 일기 쓰는 것은 좋지만 보기가 조금 불편해서 추가적인 기능들이 있으면 좋겠다 싶었음.
- 현재 쓰면서 느낀 불편한 점 :
  한 사람의 일기 모아보기 기능
  캘린더에서 보는 기능
  일기 들어갔다 나오면 스크롤 초기화 돼서 맨 위로 올라감
  작성 중 임시저장기능
  프로필 사진 설정은 있지만 일기장 내에선 볼 수 없음 등등

## 현재 앱에 있는 기능
- 로그인, 회원가입, 프로필 수정
- 일기 작성, 사진 추가, 수정, 좋아요, 댓글 작성
- 친구를 초대해서 하나의 다이어리에서 함께 일기 작성
- 모든 일기 리스트로 표시
- 일기장 상단에 공지 한줄 정도 작성 가능
- 감정 이미지 선택 가능

## 추가해보고 싶은 기능
- 달력 띄워서 날짜 클릭 시 해당 날짜 일기들 뜨게하기
- 한 사람의 일기 모아보기 기능
- 일기 보려고 들어갔다 나오면 스크롤 초기화 돼서 맨 위로 올라감 -> 스크롤 위치 기억
- 일기 작성 시 임시 저장 기능
- 프로필 사진 보여주기
- 자신의 일기에 댓글 혹은 좋아요 누르면 알람 기능
